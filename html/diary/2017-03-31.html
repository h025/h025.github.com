git简介

Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件

git的基本命令

git help: 显示git的基本命令
git init：初始化本地仓库
git add fileName.fileType:将工作文件修改提交到本地暂存区
git add . : 将所有修改过的工作文件提交暂存区
git commit -m “文件描述” :文件提交至本地
git log : 查看该文件每次提交记录
git push origin master : 把本地文件提交至指定服务器

git基本运用

1.登录github。 2.创建项目，并复制秘钥 3.克隆项目：本地打开dos窗口，输入git cione 秘钥地址 4.本地添加项目文件至克隆文件夹内 5.cd fileName:进入克隆项目文件夹内 6.git add . ：添加所有文件进入提交暂存区 7.git commit _m "" :把文件提交至本地 8.git push origin master:文件提交至网络 9.输入用户名 10.输入用户密码（注意；密码是不显示的）
